Optional resolution

resolution: string
  • Defined in RecordingProperties.ts:58

Recording video file resolution. Must be a string with format "WIDTHxHEIGHT", being both WIDTH and HEIGHT the number of pixels between 100 and 1999.
Will only have effect if RecordingProperties.outputMode is set to Recording.OutputMode.COMPOSED. For Recording.OutputMode.INDIVIDUAL all individual video files will have the native resolution of the published stream
